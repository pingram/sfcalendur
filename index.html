SF Calendur

What?

A collaborative calendar to ensure no one misses another event with notifications when you need to start organizing, NOT just when the event actually occurs.

Why?

Too many times you hear about a cool band coming to town, a comedian, BYOBW, etc. too late to get tickets / organize.

How?

  1. Add sfcalendur@gmail.com to your contacts
  2. Create 2 events on your calendar and invite sfcalendur@gmail.com: the first the day of the event, the second on the organizing start date
  3. I will review the event (and its organizing start date event) and accept it if it meets the event creation requirements
  4. I will remove your personal information and add it to sfcalendur's Google Calendar
  5. Other's will see your event and organize their friends!


Organizing Start Date

The organizing start date (OSD) is the date on which people will need to start organizing their friends. It should typically be 2 weeks prior to any deadlines.

For example, let's say you want to add Outside Lands 2015 to the calendar. The actual event is August 7-9, but tickets go on sale April 2. The OSD is 2 weeks prior to April 2, or March 19.

Event Creation Requirements

All events must have the following:

  1. A description including any appropriate links
  2. An accompanying OSD event with the same name as the event but with " OSD" appended
